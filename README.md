# ĀDI — From Nothing. To Everything.

A cinematic, scroll-driven interactive journey through **13.8 billion years** of creation — built with Three.js, custom GLSL shaders, and GSAP ScrollTrigger.

## Experience
Witness the birth of the universe, the formation of stars, the rise of Earth, and the evolution of life — culminating in a glimpse of an unknown future. Every scroll equals millions of years.

## Tech Stack
- **Three.js r165** — WebGL2 with multi-pass render targets
- **Vite 5** — with `vite-plugin-glsl` for shader imports
- **GSAP 3** — ScrollTrigger for camera path animation
- **Howler.js** — Spatial ambient audio per era
- **100% Custom GLSL** — Explosion, planet, atmosphere, singularity shaders

## Getting Started
```bash
npm install
npm run dev
```
